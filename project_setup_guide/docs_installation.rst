Installation
============

Using pip
---------

.. code-block:: bash

    pip install {{PROJECT_NAME}}

From source
-----------

.. code-block:: bash

    git clone https://github.com/{{GITHUB_USERNAME}}/{{PROJECT_NAME}}.git
    cd {{PROJECT_NAME}}
    pip install -e .[dev]

Using uv
--------

.. code-block:: bash

    uv sync
