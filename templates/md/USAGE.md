# Using This Template

## For Template Users

This is a **template repository**. To use it:

1. **Run the setup script**:
   ```bash
   python setup_project.py
   ```

2. **Verify the setup**:
   ```bash
   python scripts/validate_project.py
   ```

3. **Start developing**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -e .[dev]
   make test
   ```
